# URL Shortener Service

## Обзор сервиса

Сервис для создания коротких ссылок с возможностью управления, аналитики
и кастомными алиасами. Позволяет сокращать длинные URL, отслеживать
статистику переходов и управлять временем жизни ссылок.

### Ключевые возможности

-   **Создание коротких ссылок** --- автоматическая генерация или
    кастомные алиасы\
-   **Управление ссылками** --- полный CRUD для коротких ссылок\
-   **Статистика** --- отслеживание количества переходов, даты создания
    и последнего использования\
-   **Временные ссылки** --- автоматическое удаление по истечении срока
    действия\
-   **Поиск** --- поиск коротких ссылок по оригинальному URL

------------------------------------------------------------------------

## Технологический стек

-   **FastAPI** --- веб‑фреймворк\
-   **SQLAlchemy 2.0** --- асинхронный ORM\
-   **SQLite** --- база данных\
-   **APScheduler** --- планировщик задач\
-   **Pydantic** --- валидация данных

# Эндпоинты API

  Метод    Эндпоинт                    Описание
  -------- --------------------------- -------------------------------------
  GET      /                           Корневой эндпоинт с документацией
  GET      /health                     Проверка работоспособности
  POST     /links/shorten              Создание короткой ссылки
  GET      /links/{short_code}         Перенаправление на оригинальный URL
  GET      /links/{short_code}/info    Информация о короткой ссылке
  PUT      /links/{short_code}         Обновление оригинального URL
  DELETE   /links/{short_code}         Удаление короткой ссылки
  GET      /links/{short_code}/stats   Статистика по ссылке
  GET      /links/search               Поиск по оригинальному URL

# Установка и запуск

## 1. Клонирование репозитория

``` bash
git clone <repository-url>
cd url-shortener
```

## 2. Создание виртуального окружения

``` bash
# Создание виртуального окружения
python -m venv venv

# Активация на macOS/Linux
source venv/bin/activate

# Активация на Windows
venv\Scripts\activate
```

## 3. Установка зависимостей

``` bash
pip install -r requirements.txt
```

## 4. Запуск сервиса

``` bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

  -----------------------------------------------------------------------
  Компонента              Что делает              Для чего нужно
  ----------------------- ----------------------- -----------------------
  uvicorn                 ASGI-сервер             Запускает FastAPI
                                                  приложение

  main:app                Путь к приложению       main = файл main.py /
                                                  app = переменная
                                                  FastAPI

  --reload                Автоперезагрузка        Сервер перезапускается
                                                  при изменении кода

  --host 0.0.0.0          Привязка к адресу       Делает сервер доступным
                                                  по локальной сети

  --port 8000             Порт сервера            Стандартный порт
                                                  разработки
  -----------------------------------------------------------------------

Альтернативный запуск через Docker Compose: 
``` bash
# Собрать и запустить контейнер
docker-compose up --build

# Запустить в фоновом режиме
docker-compose up -d

# Остановить контейнер
docker-compose down

# Посмотреть логи
docker-compose logs -f
```

**Доступ:**\
http://localhost:8000

## 3. POST /links/shorten --- Создание короткой ссылки

### Вариант 1: Автоматическая генерация

``` json
{
  "original_url": "https://example.com/very/long/url/that/needs/shortening"
}
```

**Ответ:**

``` json
{
  "original_url": "https://example.com/very/long/url/that/needs/shortening",
  "short_code": "aB3dE5",
  "short_url": "http://localhost:8000/links/aB3dE5",
  "created_at": "2024-01-01T12:00:00.123456",
  "expires_at": null,
  "is_custom": false
}
```

### Вариант 2: Кастомный alias

``` json
{
  "original_url": "https://example.com/about",
  "custom_alias": "about-us",
  "expires_at": "2024-12-31T23:59:00"
}
```
